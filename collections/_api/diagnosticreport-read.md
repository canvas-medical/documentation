This endpoint implements the US Core DiagnosticReport Profile for Report. The following USCDI data elements are retrievable from this endpoint:

Clinical Notes: 

* Imaging Narrative
* Laboratory Report Narrative
* Pathology Report Narrative
* Procedure Note

Laboratory:

* Tests
* Values/Results
